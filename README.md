Installation

To run this project locally:

1.	Clone the Repository:

         https://github.com/seriuksergii/TaskTracker-Vue-.git

2.	Navigate to the project directory:

         cd TaskTracker-Vue-

3.	Install dependencies:

          npm install

4.	Start the application:
 
          npm start

•	Vue

•	TypeScript
